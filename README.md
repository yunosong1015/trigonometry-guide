# Project: Trigonometry Guide (삼각비 가이드)

## 1. 개요
중학교 수준의 삼각비(sin, cos, tan)를 쉽게 배울 수 있는 깔끔하고 현대적인 웹사이트를 구축합니다.

## 2. 주요 기능 및 디자인
- **Hero Section**: 삼각비의 정의와 흥미를 유발하는 제목.
- **Definition Section**: 직각삼각형에서의 sin, cos, tan 정의를 시각적으로 설명 (SVG 사용).
- **Interactive Calculator (Optional)**: 각도를 입력하면 삼각비 값을 계산해주는 간단한 도구.
- **Table Section**: 주요 각도(0, 30, 45, 60, 90)에 대한 삼각비 표.
- **Responsive Design**: 모바일과 데스크톱 모두에서 잘 보이도록 설계 (Vanilla CSS).

## 3. 기술 스택
- **HTML5**: 구조 정의.
- **CSS3 (Vanilla CSS)**: 현대적인 레이아웃 및 스타일링 (Flexbox/Grid).
- **JavaScript (Vanilla JS)**: 간단한 인터랙션 및 계산 기능.

## 4. 파일 구조
- `index.html`: 메인 페이지.
- `style.css`: 스타일 시트.
- `script.js`: 인터랙션 로직.
